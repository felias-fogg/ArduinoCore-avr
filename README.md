This branch of the repository contains, as demo software, the source code and configuration files of a debug-enabled Arduino AVR Boards [platform](https://arduino.github.io/arduino-cli/latest/platform-specification/). Documentation can be found at [https://pyavrocd.io](https://pyavrocd.io). Try at your own risk by installing the following board manager URL: 

```https://felias-fogg.github.io/ArduinoCore-avr/package_felias-fogg_ArduinoCore-avr_index.json```
